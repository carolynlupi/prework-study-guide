HTML

  • The head element contains information about the webpage.
  • The body elemtn represents the visable content shown to the user.
  • Elements that are nested inside of other elements are called child elements.

CSS

  • CSS stands for Cascading Style Sheets.
  • CSS is the language used to describe the presentation of a website thats created with HTML.

Git

  • Git commands always start with "git".
  • Git is a free, open source version control system that allows web developers to collaborate on projects without overwriting each other's work.
  • It also allows developers to revert to previous versions of the code if a problem is discovered.

JavaScript

  • Javascript is a lightweight programming language that is used to provide logic and interactibity on websites.
